Josie ML was created as a project for the Major League Hacking's AI/ML Global Hack Week Hackathon (February 9-15, 2024).

Josie ML's Goal:

To give the user an option to either submit an image or a drawing of a number or an object and use the trained & tested machine learning algorithm to correctly identify the input.

Creator's Goal:

To practice implementing a machine learning algorithm and combining it with a frontend web user interface using Python & Django.
